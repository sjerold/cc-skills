# 衔风搜索 - 需求文档 (修订版)

## 1. 概述
- **目标**: 开发一个飞书云文档（私有化部署）搜索插件，支持目录遍历、缓存、文档名匹配、内容抓取
- **背景**: 用户需要快速搜索衔风云文档（飞书私有化部署）中的文档，并能抓取文档内容

## 2. 系统说明

### 2.1 目标系统
- **名称**: 衔风云文档
- **类型**: 飞书云文档私有化部署版本
- **访问方式**: 网页浏览器访问
- **登录要求**: 需要登录

### 2.2 文档类型
- 在线文档（类似Google Docs / Notion）
- 内容需要从网页抓取

## 3. 功能需求

### 核心功能 (必须实现)

- [ ] **登录处理**: 打开浏览器，等待用户手动登录飞书
- [ ] **目录遍历**: 遍历飞书云文档的完整目录结构（知识库/文件夹层级）
- [ ] **目录缓存**: 将目录结构缓存到本地JSON文件
- [ ] **文档名搜索**: 根据关键词匹配文档名称
- [ ] **内容抓取**: 使用Playwright抓取在线文档内容并保存为MD
- [ ] **结果整理**: 整合搜索结果，返回给用户

### 扩展功能 (可选)

- [ ] **增量缓存更新**: 检测目录变化，只更新变化的部分
- [ ] **全文搜索**: 支持在文档内容中搜索（而非仅文档名）
- [ ] **批量导出**: 批量导出多个文档内容

## 5. 非功能需求

- **性能要求**:
  - 目录遍历支持大型知识库（1000+文档）
  - 缓存加载 < 1秒
  - 文档名匹配 < 500ms
  - 单文档抓取 < 10秒

- **安全要求**:
  - 不保存用户登录凭据
  - 缓存文件权限控制

- **兼容性要求**:
  - Windows 10/11
  - Chrome浏览器
  - 复用百度搜索的Playwright环境

## 6. 验收标准

- [ ] 能够打开飞书网页并等待用户登录
- [ ] 能够遍历飞书云文档目录并生成缓存
- [ ] 能够从缓存中搜索文档名匹配的结果
- [ ] 能够打开文档页面并抓取内容
- [ ] 能够将抓取内容保存为MD文件
- [ ] 输出格式化的搜索结果（JSON）

## 7. 边界情况

| 情况 | 处理方式 |
|------|----------|
| 登录超时 | 提示用户，退出程序 |
| 目录遍历失败 | 记录错误，继续尝试 |
| 文档无法访问 | 记录错误，跳过该文档 |
| 内容抓取失败 | 记录错误，返回部分结果 |
| 搜索无结果 | 返回空列表，提示建议 |
| 网络中断 | 重试3次，记录错误 |

## 8. 技术约束

### 必须使用的技术/框架
- Python 3.10+
- Playwright (复用百度搜索配置)
- Chrome浏览器

### 参考实现
- 百度搜索插件: `~/.claude/plugins/baidu-search/`

## 9. 插件结构

```
~/.claude/plugins/xianfeng-search/
├── .claude-plugin/
│   └── plugin.json           # 插件配置
├── skills/
│   └── xianfeng-search/
│       └── SKILL.md          # 技能说明
├── scripts/
│   ├── xianfeng_search.py    # 主搜索脚本
│   ├── feishu_navigator.py   # 飞书导航模块
│   ├── directory_scanner.py  # 目录扫描模块
│   ├── cache_manager.py      # 缓存管理模块
│   └── content_fetcher.py    # 内容抓取模块
├── cache/                    # 缓存目录
│   └── *.json               # 目录结构缓存
└── requirements.txt          # 依赖
```

## 10. 命令接口

```bash
# 基本搜索
python xianfeng_search.py "关键词"

# 指定飞书域名
python xianfeng_search.py "关键词" --domain "https://your-feishu.example.com"

# 强制刷新缓存
python xianfeng_search.py "关键词" --refresh

# 显示浏览器（用于手动操作）
python xianfeng_search.py "关键词" --show-browser

# 仅遍历目录（不搜索）
python xianfeng_search.py --scan-only

# 显示缓存状态
python xianfeng_search.py --cache-status

# 清理缓存
python xianfeng_search.py --clear-cache
```
