# 公文格式详细规范

本文档提供党政机关公文格式的完整参考规范。

## 字体对照表

| 字号 | 磅值 | 用途 |
|------|------|------|
| 二号 | 22磅 | 公文标题 |
| 三号 | 16磅 | 正文、发文字号、主送机关等 |
| 四号 | 14磅 | 版记、页码 |
| 小四 | 12磅 | 附件说明 |

## 常用字体

| 字体名称 | 用途 |
|----------|------|
| 方正小标宋简体 | 公文标题 |
| 仿宋_GB2312 | 正文、发文字号、署名、日期 |
| 黑体 | 一级标题、密级、附件标识 |
| 楷体_GB2312 | 二级标题 |
| 宋体 | 页码 |
| Times New Roman | 数字、年份 |

## 标题层级规范

### 一级标题
- 格式: `一、标题内容`
- 字体: 3号黑体
- 编号: 中文数字 + 全角顿号
- 示例: `一、培训目标`

### 二级标题
- 格式: `（一）标题内容`
- 字体: 3号楷体_GB2312（不加粗）
- 编号: 中文数字加括号
- 示例: `（一）业务知识培训`

### 三级标题
- 格式: `1. 标题内容`
- 字体: 3号仿宋_GB2312（加粗）
- 编号: 阿拉伯数字 + 点
- 示例: `1. 风险防控培训`

### 四级标题
- 格式: `（1）标题内容`
- 字体: 3号仿宋_GB2312
- 编号: 阿拉伯数字加括号
- 示例: `（1）信贷风险`

## 发文字号格式

```
格式: 发文机关代字〔年份〕顺序号
示例: 苏银发〔2026〕15号

要点:
1. 年份用六角括号〔〕，不用方括号[]
2. 顺序号不加"第"字
3. 不编虚位（1不写成01）
4. 后加"号"字
```

## 常见公文文种

| 文种 | 用途 | 特点 |
|------|------|------|
| 决定 | 重要事项安排部署 | 权威性、约束力强 |
| 命令(令) | 公布法规、任免等 | 最高权威性 |
| 公报 | 公布重大决定事项 | 公开性 |
| 公告 | 向社会宣布重要事项 | 公开、广泛 |
| 通告 | 公布社会周知事项 | 强制性 |
| 意见 | 提出见解和处理办法 | 指导性 |
| 通知 | 传达事项、部署工作 | 最常用 |
| 通报 | 表彰先进、批评错误 | 教育、警示 |
| 报告 | 向上级汇报工作 | 陈述性、不要求回复 |
| 请示 | 向上级请求指示 | 请求性、必须回复 |
| 批复 | 答复下级请示 | 针对性、权威性 |
| 议案 | 提请审议事项 | 法定性 |
| 函 | 商洽工作、询问答复 | 商洽性 |
| 纪要 | 记载会议主要情况 | 纪实性 |

## 标点符号使用规范

- 主送机关后用全角冒号：
- 附件后用全角冒号：
- 抄送后用全角冒号：
- 同系统内同级机关用顿号、
- 不同系统机关用逗号，
- 最后一个抄送机关后用句号。
- 标题末尾一般不加标点
- 附件名称后不加标点
