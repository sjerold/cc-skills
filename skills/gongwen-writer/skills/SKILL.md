# 公文写作技能

本技能用于指导创建符合党政机关公文格式规范的Word文档。

## 适用场景

- 撰写党政机关公文
- 创建规范化Word文档
- 起草通知、报告、请示等公文

## 公文格式规范要点

### 一、公文用纸
- **纸张规格**: 国际标准A4型 (210mm × 297mm)
- **纸张定量**: 60g/m² ～ 80g/m² 胶版印刷纸或复印纸
- **纸张白度**: 80% ～ 90%
- **横向耐折度**: ≥ 15次
- **不透明度**: ≥ 85%
- **PH值**: 7.5 ～ 9.5

### 二、排版规格

**页边距**:
- 上下页边距: 36mm
- 左右页边距: 27mm
- 版心尺寸: 156mm × 225mm (不含页码)

### 三、份号、密级和保密期限

**份号**:
- 6位3号阿拉伯数字
- 顶格编排于版心左上角第一行

**密级和保密期限**:
- 3号黑体字
- 顶格标于版心左上角第二行
- 保密期限用阿拉伯数字
- 密级和保密期限用"★"分隔
- 只标密级不标期限时，"秘密"两字间空1字

### 四、公文版头(发文机关标志)

- 由发文机关全称或规范化简称后加"文件"组成
- 特定公文可只标识发文机关全称或规范化简称
- 版头上边缘至上页边距: 50mm
- 发文机关名称字号: 小于22mm × 15mm

### 五、发文字号

- 组成: 发文机关代字 + 发文年份 + 发文顺序号
- 位置: 发文机关标识下空二行，居中排布
- 字体: 3号仿宋_GB2312体字
- 年份、发文顺序号: 阿拉伯数字(Times New Roman)体
- 年份用六角括号"〔〕"括入
- 发文顺序号不加"第"，不编虚位，后加"号"字
- 示例: `苏银发〔2026〕15号`

### 六、红色分隔线

- 带有"文件"2字的版头: 红色五角星加两侧红色横线
- 位置: 与发文字号底部相距4mm
- 长度: 与版心等宽
- 高度: 0.35mm ～ 0.5mm

### 七、公文标题

- 组成: 发文机关全称/规范化简称 + 公文主题 + 文种
- 位置: 红色分隔线下空一行
- 字体: 2号方正小标宋简体
- 排列: 一行或多行居中排布
- 回行要求: 词意完整，排列对称，间距恰当

### 八、主送机关

- 位置: 标题下空一行，左侧顶格
- 字体: 3号仿宋_GB2312体字
- 回行时仍顶格
- 最后一个主送机关名称后标全角冒号

### 九、公文正文

**基本格式**:
- 字体: 3号仿宋_GB2312体字
- 每页: 22行
- 每行: 28个字
- 行距: 固定值28磅

**标题层级格式**:

| 级别 | 编号格式 | 字体 | 示例 |
|------|----------|------|------|
| 一级标题 | 中文"一" + 全角顿号 | 3号黑体 | 一、标题 |
| 二级标题 | 中文"(一)" | 3号楷体_GB2312(不加粗) | (一)标题 |
| 三级标题 | 阿拉伯数字 + 点 | 3号仿宋_GB2312(加粗) | 1. 标题 |
| 四级标题 | (1) | 3号仿宋_GB2312 | (1)标题 |

**注意**: 数字、年份不回行

### 十、公文附件

**附件标注**:
- 位置: 正文下空一行，左空二字
- 字体: 3号仿宋_GB2312体字
- 格式: "附件"后标全角冒号和名称
- 附件名称回行: 与上一行附件名称首字对齐
- 多个附件: 用阿拉伯数字标明顺序，如"附件：1. ×××"
- 附件名称后不加标点符号

**附件本体**:
- 与正文一起装订
- 版心左上角第一行用3号黑体标识"附件"
- 后不标全角冒号

**附注**:
- 字体: 3号仿宋_GB2312体字
- 位置: 左空二字加圆括号，成文日期下一行
- 示例: "此件公开发布"、"此件不予公开"
- 请示件: 注明联系人及联系电话

### 十一、发文机关署名

- 用发文机关全称或规范化简称
- 字体: 3号仿宋_GB2312体字
- 位置: 正文右下方，以成文日期为准居中编排
- 上距正文: 二至三行

### 十二、成文日期

- 定义: 会议通过日期或领导人签发日期
- 多位领导审签: 署最后一位签发日期
- 联合行文: 署最后签发机关领导人签发日期
- 字体: 3号仿宋_GB2312体字
- 格式: 阿拉伯数字年、月、日标全
- 加盖印章公文: 右空4字编排
- 不加盖印章公文: 以发文机关首字为准右空2字编排
- 无主送机关公文: 加圆括号标于标题下方居中

### 十三、印章

- 适用: 除印有固定版头的普发性文件和会议纪要外
- 位置: 成文日期中间偏上
- 要求: 上不压正文、下要骑年盖月
- 当前页无正文: 页首加圆括号标识"此页无正文"

### 十四、版记

**印制版记组成**: 印发机关名称、印发日期、抄送机关

**印发机关名称**:
- 一般为机关办公室，也可由发文机关担任
- 位置: 下左侧空一字
- 字体: 4号仿宋_GB2312体字

**印发日期**:
- 位置: 右侧空一字
- 字体: 4号仿宋_GB2312体字
- 年月: 阿拉伯数字(Times New Roman)体

**抄送机关**:
- 范围: 除主送机关外需告知内容的上级、下级和无隶属关系机关
- 顺序: 先上级后平级再下级
- 标点: 同系统内同级机关用顿号，不同系统用逗号，最后用句号
- 位置: 版记左上方空一字
- 标识: "抄送"后标全角冒号和抄送机关
- 字体: 4号仿宋_GB2312体字
- 回行: 与冒号后的抄送机关对齐

### 十五、页码

- 字体: 4号宋体
- 格式: 阿拉伯数字，左右各放一条一字线
- 位置: 公文版心下边缘之下居中
- 一字线距版心下边缘: 约7mm
- 附件与正文一起装订时: 页码连续编排

### 十六、装订要求

- 方式: 左侧装订
- 骑马订/平订订位: 两钉外订眼距版面上下边缘各70mm处
- 平钉钉距与书脊间距: 3mm ～ 5mm

### 十七、阿拉伯数字字体规范

**重要规则**: 公文中所有阿拉伯数字必须使用 **Times New Roman** 字体

适用范围:
- 发文字号中的年份、顺序号
- 份号
- 成文日期
- 页码
- 正文中的数字
- 附件序号
- 印发日期
- 统计数据、金额、百分比等

## Word文档实现要点

使用python-docx库创建公文时，需要设置:

```python
from docx import Document
from docx.shared import Pt, Mm, Cm
from docx.enum.text import WD_ALIGN_PARAGRAPH
from docx.oxml.ns import qn

# 页面设置
sections = document.sections
for section in sections:
    section.page_height = Mm(297)      # A4高度
    section.page_width = Mm(210)       # A4宽度
    section.top_margin = Mm(36)        # 上边距
    section.bottom_margin = Mm(36)     # 下边距
    section.left_margin = Mm(27)       # 左边距
    section.right_margin = Mm(27)      # 右边距

# 行距设置
paragraph.paragraph_format.line_spacing = Pt(28)

# 中文字体设置
run.font.name = '仿宋_GB2312'
run._element.rPr.rFonts.set(qn('w:eastAsia'), '仿宋_GB2312')
run.font.size = Pt(16)  # 3号字约16磅

# 阿拉伯数字字体设置 (Times New Roman)
run.font.name = 'Times New Roman'
run.font.size = Pt(16)
```

- 命令(令)

- 公告
- 通告
- 意见
- 通知
- 通报
- 报告
- 请示

- 议案
- 函
- 纪要
