# 百度搜索增强版 (baidu-search)

大规模搜索、智能分数筛选、内容抓取、自动总结。

## 功能特性

- **大规模搜索** - 支持获取多达 50+ 条搜索结果
- **智能评分** - 基于正则匹配的质量评分矩阵，识别来源、关键词匹配、内容类型
- **内容抓取** - 使用 Playwright 动态渲染，支持 JS 页面
- **自动总结** - 生成 Markdown 格式的搜索报告和总结
- **Session 管理** - 每次搜索独立目录，便于管理
- **智能过滤** - 自动跳过视频网站(B站、YouTube等)、图片网站、下载站

## 评分机制

搜索结果按 0-10 分评估质量：

| 来源类型 | 分数范围 |
|---------|---------|
| 官方文档 (Python.org, GitHub) | 9.0-9.6 |
| Stack Overflow | 9.5 |
| SegmentFault | 8.0-9.4 |
| CSDN | 7.5-8.6 |
| 知乎 | 7.0-8.4 |
| 菜鸟教程 | 6.5-7.6 |
| 视频教程 (B站) | 4.0-7.6 |
| 百度文库/知道 | 3.5 |

评分维度：
1. **来源质量** - 从标题/摘要识别官方文档、技术社区
2. **关键词匹配** - 标题和摘要与搜索词的相关性
3. **内容类型** - 教程、指南、问题解答等高价值内容加分
4. **过滤项** - 广告下载、营销推广等低质量内容减分

## 使用方法

```bash
# 基本搜索
python baidu_search.py "搜索关键词"

# 指定结果数量
python baidu_search.py "搜索关键词" -n 10

# JSON 输出
python baidu_search.py "搜索关键词" --json
```

## 版本历史

### v2.6.0 (2026-04-20)
- 修复知乎文章抓取（解决"安全验证"误判为反爬的问题）
- 新增视频/图片/下载网站自动跳过（B站、YouTube、百度图片、CSDN下载等）
- 清理特殊Unicode字符，避免编码错误

### v2.5.0 (2026-04-08)
- 新增 `scorer.py` 独立评分模块，遵循 Clean Code 原则
- 重构打分机制：正则匹配评分矩阵替代原有域名匹配
- 支持从标题/摘要识别来源质量（解决百度跳转链接无法识别域名问题）
- 新增关键词匹配度加分机制
- 新增内容类型判断（教程、指南、问题解答等）
- 移除 AI 评分依赖，提升稳定性和响应速度

### v2.4.0 (2026-04-03)
- 添加应用下载链接过滤（应用宝、Google Play、软件下载站等）
- 默认并发数改为 4，提升抓取效率
- 优化搜索结果质量筛选

### v2.3.0
- 改进跳转链接检测逻辑
- 使用 wait_for_selector 优化页面加载等待

### v2.2.0
- 全部改为 Playwright async API
- 分离 content_parser 和 markdown_writer 模块
